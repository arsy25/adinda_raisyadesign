3CSS Structure

Let's take a look closely at our stylesheet (css) files:

  • bootstrap.min.css - standard bootstrap css styles.
  • iconfont.css - contain css styles for iconfont icons.
  • responsive.css - common css responsive styles for this template.
  • main.css - css styles for this template.
  • owl.carousel.min.css- owl carousel
  • owl.theme.default.min.css- owl carousel
  • dark.css- dark style for this themes
  • magnific.css- magnific css

And css for dark themes

4JavaScripts

This template imports Javascript files. jQuery is a Javascript library that greatly reduces the amount of code that you must write. All the Javascript files can be found inside the javascript folder.
